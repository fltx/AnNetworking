# AnNetworking
python自动生成工程网络框架

步骤1.cd 到AnNetworking目录下</br>
步骤2.执行命令:</br>
  `python xxnetworking.py -n EE -f /Users/anuser/Desktop` </br>
    （EE: 网络框架的文件名和内容的前缀,/Users/anuser/Desktop: 框架输出目录） </br>
